# SmartLot 🚗

SmartLot is a smart parking system designed to help students find available parking on campus in real time.

## Features
- Real-time parking updates
- Live navigation to open spots
- Mobile-friendly design

## Problem
Students waste time searching for parking and risk being late to class.

## Solution
SmartLot provides a live map showing available parking spaces instantly.

## Slogan
"Stop circling. Start parking."
